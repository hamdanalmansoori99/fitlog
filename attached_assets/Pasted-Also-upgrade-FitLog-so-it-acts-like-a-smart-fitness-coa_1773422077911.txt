SMART COACH SYSTEM

Add a smart workout coach and recommendation engine.

Main idea:
When the user goes to the Workouts section, the app should not only let them log workouts. It should also help them decide what workouts make sense based on:
- their available equipment
- where they train
- their goals
- their experience level
- how many days per week they want to train
- how much time they have
- what they already trained recently

SMART RECOMMENDATIONS

Add a section on the Workouts page called:
"Recommended for you"

The app should generate workout recommendations based on:
- available equipment
- workout location
- goals
- preferred duration
- experience level
- recent workout history
- user preferences

Recommendation rules:
- If the user has no equipment, prioritize:
  - bodyweight workouts
  - calisthenics
  - walking
  - running
  - yoga
  - mobility
  - jump rope if available
- If the user has dumbbells, recommend dumbbell-based workouts
- If the user has barbell and bench, recommend barbell strength workouts
- If the user has gym access, recommend more complete gym workouts
- If the user has cardio equipment, include cardio options
- If the user has tennis racket or swimming pool, include tennis and swimming as options

Each recommendation card should show:
- workout name
- estimated duration
- difficulty
- equipment needed
- primary goal
- short explanation of why this workout is good for the user

Example card:
Dumbbell Upper Body Strength
- Duration: 45 min
- Difficulty: Intermediate
- Equipment: Dumbbells, Bench
- Goal: Build muscle
- Why this is good for you: Great for upper-body strength and hypertrophy using the equipment you already own.

If the user has no equipment, example:
Beginner Bodyweight Full Body
- Duration: 20 min
- Difficulty: Beginner
- Equipment: None
- Goal: Stay active / build foundational strength
- Why this is good for you: Uses bodyweight movements to improve strength, coordination, and fitness without requiring any equipment.

TODAY’S SMART SUGGESTION

Add a dashboard section called:
"Today's recommended workout"

This should use:
- user goals
- user equipment
- available workout duration
- recent training history

Example logic:
- if user trained legs yesterday, do not recommend another hard leg workout today
- if user has not trained for several days, recommend an easy restart session
- if user is consistent, recommend progression
- if user prefers short workouts, recommend a short workout first

EXERCISE MATCHING TO EQUIPMENT

When suggesting gym or strength workouts, only suggest exercises that match the user’s equipment.

Examples:
- If user has dumbbells, include:
  - dumbbell press
  - dumbbell row
  - goblet squat
  - curls
  - lunges
- If user has no barbell, do not recommend:
  - barbell squat
  - barbell deadlift
- If user has pull-up bar, include:
  - pull-ups
  - chin-ups
  - hanging leg raises
- If user has resistance bands, include:
  - band rows
  - band presses
  - band pull-aparts

SMART EXERCISE SUBSTITUTIONS

If required equipment is unavailable, automatically suggest alternatives.

Examples:
- Barbell squat -> goblet squat or bodyweight squat
- Bench press -> push-ups or dumbbell floor press
- Lat pulldown -> pull-ups or band pulldown
- Leg press -> split squat or step-ups

Show a label:
"Alternative for your setup"

But also add smart coaching behavior so the app helps the user know what to do next.

ADAPTIVE RECOMMENDATION LOGIC

Add basic smart adaptation:
- if user trained same muscle group yesterday, reduce chance of repeating it
- if user misses several days, suggest a lighter restart
- if user frequently logs a certain activity, incorporate it into future recommendations
- if user goal is weight loss, prioritize calorie-burning + sustainable sessions
- if user goal is muscle building, prioritize resistance training
- if user goal is flexibility, prioritize yoga and mobility
- if user goal is endurance, prioritize running, cycling, swimming, cardio progression

This feature should feel like a coach inside the app.

The Workouts section should include:
- Recommended for you
- Today’s recommended workout
- Your saved workout plans
- Quick log workout
- Browse workout templates

Tone should feel supportive, intelligent, and practical.

Final requirement:
Do not make FitLog feel like only a tracker.
Make it feel like a personalized fitness coach that uses the user’s equipment, goals, preferences, and history to recommend useful workouts and explain why they are good for the user.
